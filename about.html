About This Project

Project Overview

This Doppler Effect Simulation website demonstrates both the sound Doppler effect and the relativistic light Doppler effect using accurate physics formulas, real-time animations, and a clean UI. The simulator generates audio and visuals on the fly to help users explore how frequency and wavelength change with motion.

What You Can Learn

The Doppler Simulator is an interactive educational tool that highlights:

  • Sound Doppler: Real-time frequency shifts based on radial velocity
  • Light Doppler: Relativistic wavelength shifts and visual color changes
  • Interactive controls to adjust speeds, frequencies, and observation geometry
  • Clear visualizations using HTML5 Canvas and generated audio using Web Audio API

How It Works

The application uses plain HTML, CSS, and JavaScript. Visuals are drawn on the HTML5 Canvas and audio is synthesized with the Web Audio API. Physics formulas are applied in real time to animate and sonify Doppler shifts.

Mobile Compatibility

The site works on modern mobile browsers and can be opened in editors that support local live preview. Offline usage is possible once the files are downloaded to a device.
